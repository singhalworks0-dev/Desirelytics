# React + Vite + Tailwind CSS

This project is a React app built with Vite and styled with Tailwind CSS v4.

## Scripts

- `npm run dev` starts the local dev server.
- `npm run build` creates a production build.
- `npm run preview` previews the production build locally.
- `npm run lint` runs ESLint.

## Notes

- The app uses `src/main.jsx` as the entry file.
- Tailwind is loaded through `src/index.css` and the Vite Tailwind plugin.
- TypeScript config files were removed as part of the JavaScript-only cleanup.
